Detailed Description

The Text class.

This class extends the element class and is used for text that is placed on the canvas.

Author
Jonathan Kwok

This is an object for storing a bit of text that will appear on screen. It just contains the location and the text and the font.

Parameters

newLocationYThe Y coordinate where the text will appear on screen.
