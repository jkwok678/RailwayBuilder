◆ getLinkAt()

bool Track::getLinkAt ( int  link)

Gets whether there is a link a location on track.

A track has 8 possible locations to link from. 0 1 2 3 4 5 6 7 8 The links are 0, 1, 2, 3, 5, 6, 7, 8. 4 is not used. Example 1. '-' is a straight horizontal with 3 and 5 as it's links. Example 2. '/' is a bottom left to top right track with links at 2 and 6.

Parameters
linkThe link you want to check to see if there is something connected there.
Returns
A bool that describes if there is a track at that link.
